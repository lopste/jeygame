# quickgame
a javascript game engine

---
*the documentation is not here yet sadly, when there is I will link to it.*

---
quickgame is a helper library for creating games in javascript. all you need is a single canvas, a link to the library, and your own game's script.

quickgame's functions are divided into different classes, including the game in the first place. this means you can have multiple games in 1 page with only 1 quickgame link.

**to use quickgame, you will need to have some knowledge of html and javascript.**
