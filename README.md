# Jeygame
a javascript game engine

---
*The documentation is not here yet sadly, when there is I will link to it.*

---
Jeygame is a helper library for creating games in javascript. all you need is a single canvas, a link to the library, and your own game's script.

Jeygame's functions are divided into different classes, including the game in the first place.  
  This means you can have multiple games in 1 page with only 1 Jeygame link.

**To use Jeygame, you will need to have some knowledge of HTML and JavaScript.**

## Usage (These are not the docs!)

You may use this with whatever purpose according to the liscense, except for trademarking and patenting. You are also allowed to modify it all you want, this is your engine! All I ask for is some credit, if you use anything from here.

## All the stuff it has
  
- Canvas shorthands
- Canvas 2d rendering
- Easily modify sprite appearances
- Easily bind inputs 
- You tell the engine what to do, not the other way around.
